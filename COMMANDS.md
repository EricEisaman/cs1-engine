# Commands
____
**Development Builds**

```node dev engine```

```node dev socket```

```node dev neon```

```node dev building```

```node dev cosmic```
____

**Production Builds**

```node prod engine```

```node prod socket```

```node prod neon```

```node prod building```

```node prod cosmic```
____

**NPM Publish**

```node pub engine```

```node pub socket```

```node pub server```

```node pub neon```

```node pub building```

```node pub cosmic```
____

**NPM Unpublish**

```node unpub engine <version>```

```node unpub socket <version>```

```node unpub server <version>```
